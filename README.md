# skill-rolldice
Simple opsdroid skill to roll dice

Allows you to roll a d4, d6, d8, d10, d12, d20 or d100

# Requirements
None

# Configuration
```
- name: roll-dice
  repo: https://github.com/AlmightyFuzz/skill-rolldice.git
```

# Usage
Just type roll and then the dice type. 

eg: rolld4 or rolld20
